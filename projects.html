Spaceship Titanic – Kaggle Competition

Technologies: Sklearn, Xgboost, GridSearchCV

My solution for Kaggle Spaceship Titanic challenge, predicting passenger transport status (True/False) using feature engineering and an optimized XGBoost model. By decomposing categorical features and handling missing values strategically, the model achieved top 15% performance (80% accuracy).

House Prices Prediction – Kaggle Competition

Technologies: Sklearn, Xgboost, Lightgbm, StackingCVRegressor

This project was developed for a Kaggle challenge focused on predicting house prices based on various property features. I built a regression model using feature engineering and ensemble learning techniques. The solution combines several machine learning models and was inspired by the approaches shared by top participants in the competition. Final submission ranked within the top 1% results.

Learning game strategy with Q-Learning

Technologies: Manually implemented algorithm

As part of Harvard's CS50 AI course, I implemented a reinforcement learning agent that learns to play the strategic game of Nim using the Q-learning algorithm. The agent trains by playing thousands of games. It learns to choose optimal moves and can successfully compete against a human player in real time.

Watermarker – Desktop App in Python

Technologies: Tkinter, PIL

Desktop application that allows users to add and edit watermarks on images. The app supports resizing, transparency adjustment, drag-and-drop positioning, and tiling options like straight or diagonal repeat. This project gave me hands-on experience with GUI design, image processing, and event-driven programming.

Traffic Signs Recognition

Technologies: TensorFlow, Numpy

This project implements a Convolutional Neural Network built with TensorFlow/Keras to recognize traffic signs from the GTSRB dataset. The model includes images scaling, batch normalization, and dropout layers to prevent overfitting. Achieved accuracy above 98% on a 40% test split.

Bikey weather - SMS Alerts for Cyclists

Technologies: API, Twilio.rest

Python app that sends SMS notifications when the weather is ideal for a cycling trip. By fetching data from a weather API, it checks if the upcoming weekend will have strong, steady winds and minimal chance of rain. If conditions are favorable, it sends an SMS suggesting the best cycling direction (with the wind) and a recommended destination.
